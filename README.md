# 本地视频

这是一个基于谷歌[Media3](https://developer.android.google.cn/guide/topics/media/media3)开发的开源本地视频播放软件，使用[Material Design 3](https://m3.material.io/)设计规范，意在实现轻量，美观，实用。

## 已实现功能

- 选择视频内置音轨和字幕轨道
- 手势调节（亮度，音量，进度）
- 按文件夹分类视频
- 检索本地视频
- 保存视频进度
- 批量管理视频
- 主页背景设置
- 温馨提醒
- ......
